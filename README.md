# HRSRD: High-Resolution SAR Road Dataset

Dataset repository for **HRSRD**, a high-resolution SAR dataset for road extraction.

---

## News
2026.03.13  The dataset will be publicly released upon paper acceptance.


## Overview
HRSRD is designed for research on road extraction from Synthetic Aperture Radar (SAR) imagery.  
The dataset provides SAR image patches together with corresponding road annotations.
